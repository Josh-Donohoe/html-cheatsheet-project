The Basics of HTML Code

Basic HTML Code

Tag Name Description
<h1> Header This tag is used for headings on a page. The <h1> denotes the most important heading.
<h(n)> Header (2-6) This tag is used for the remaining headings on a page. For the second most important heading <h2> should be used, through to the least important heading, where <h6> should be used.
<p> Paragraph This tag should be used for the bulk of your text. Wherever you want a paragraph, use this tag.
<!--...--> Comment This tag is used to create a comment that can only be seen in the html code, and not your live website. For example, if you needed to write a comment to other developers then you would insert this tag.
<img> Image If you want to insert an image into your web page use this tag. Remember to add an src="" property as well as an alternate text for increased accessibility. It is also possible to use the tag <figcaption> to add a caption to your image.
<video> Video Similar to the above tag but adds a video. Remember to specify controls as well as size in order to give users the best experience.

Basic Structural HTML Code

Tag Name Description
<html> HTML All your html code needs to be held within this tag.
<head> Header This tag contains the meta data of a page, including the title (not the header, but what is shown in your open tab at the top of your browser), and links to fonts and the CSS stylesheet.
<body> Body This is where the content of your webpage is held.
<footer> Footer This is used for taskbars, contact details or other information at the bottom of your page that needs to be seperate from the main content.
